# Alberta Driving Study Hub

A dependency-free static prototype for an Alberta driver's licence study hub.

Open `index.html` in a browser to use the app. Progress is saved in the browser with `localStorage`.

## Included

- Hierarchical navigation matching the requested information architecture
- Start Here pages and a licence pathway tool
- Class 7, Class 5/GDL, Class 6, and commercial licence study sections
- Shared learning centre topics
- Practice questions and progress tracking
- Official Alberta resource links
